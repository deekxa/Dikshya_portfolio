# Public Images Directory

Add your images here:

## Structure:
- `/images/profile.jpg` - Your profile photo
- `/images/projects/` - Project screenshots
- `/images/icons/` - Custom icons and logos

## Recommended Sizes:
- Profile photo: 400x400px
- Project images: 1200x630px (for good quality and performance)
- Icons: 64x64px or SVG format

Remember to optimize images before adding them to reduce file sizes.
